# Sexual Health Tariff Grouper

The Sexual Health Tariff Grouper is a SaaS web application that automates tariff implementation for sexual health services. It processes activity data submitted by providers and calculates tariff charges based on currency triggers, cross-charging rules, and market forces factors — delivering accurate, transparent costing for commissioners and providers alike.
